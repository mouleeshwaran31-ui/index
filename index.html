01 // Project Overview

What We're Building

A digital needle positioning system for pattern jacquard automates the conversion of user-designed patterns into binary signals to control pin positions on jacquard wheels.

It replaces manual pin setting with a microcontroller-based mechanism that ensures accurate, repeatable, and efficient fabric pattern production.

The Automated Pin Programming Machine is a smart automation system designed to perform accurate pin positioning and programming with minimal human effort. The system works by receiving input commands through a control interface, processes the instructions, and automatically aligns the target position with high precision.

This eliminates manual errors, improves programming speed, and ensures consistent output quality across all production cycles.

02 // Process Flow

How It Works

01
Input Command
The operator inputs programming parameters through a centralized Human-Machine Interface (HMI). Pattern data is digitized and encoded into binary control signals.
02
Intelligent Processing system
The ESP32 microcontroller processes positional coordinates, interprets binary pattern data, and generates motor control signals for precise movement execution.
03
Motor Movement
High-precision motion actuators activate and align the needle positioning mechanism to the target coordinates with maximum accuracy.
04
Pin Positioning
The system precisely positions each pin on the jacquard wheel according to the pattern data, replacing the error-prone manual pin-setting process entirely.
05
Autonomous Execution
Integrated sensors continuously monitor positioning and motion accuracy.
06
Completion
The system confirms successful pattern encoding, logs the operation, and readies itself for the next cycle — fully autonomous from start to finish.

04 // System Design

Architecture

Hardware Components
  • ESP32 Microcontroller — central processing unit
  • Stepper motors for high-torque, precise needle movement
  • Custom PCB with integrated motor driver ICs
  • 3D-printed chassis and mechanical assemblies
  • HMI interface panel for operator input
Software Architecture
  • Pattern digitalization — bitmap to binary signal conversion
  • Real-time OS task scheduling on ESP32
  • PID-based motor control loop
  • HMI communication protocol
Key Innovations
  • Fully replaces manual jacquard pin setting mechnism
  • Binary pattern encoding from graphical input
  • Closed-loop motion control for repeatability
  • Modular design for different machine compatibility
Performance Targets
  • maximum accuracy positioning accuracy
  • minimum manual intervention after pattern upload
  • Significant reduction in setup time vs manual method
